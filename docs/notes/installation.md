# Beta-recsys

Welcome to Read the Beta-RecSys docs



## how to install

How to install

```shell
$ git clone https://github.com/beta-team/beta-recsys.git
$ cd $project_path$
$ python setup.py install --record files.txt
```

How to completely uninstall

```shell
$ cd $project_path$
$ xargs rm -rf < files.txt
```
